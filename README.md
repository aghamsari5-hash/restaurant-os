# RestaurantOS

RestaurantOS is a production-ready, enterprise-grade Restaurant Management Platform built using modern software architecture principles.

## Status

🚧 In Development

## Architecture

- Clean Architecture
- Domain Driven Design (DDD)
- Modular Monolith
- Event Driven Architecture
- Multi-Tenant

## Tech Stack

### Backend

- NestJS
- Prisma
- PostgreSQL
- Redis
- Swagger

### Frontend

- Next.js
- React
- TypeScript
- TailwindCSS
- shadcn/ui
- TanStack Query
- Zustand

### Infrastructure

- Docker
- Docker Compose
