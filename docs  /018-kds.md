# RestaurantOS Kitchen Display System (KDS)

Version: 1.0

---

# Goal

The Kitchen Display System (KDS) replaces traditional kitchen printers.

Orders are displayed in real time.

Kitchen staff can manage preparation without relying on paper.

The system must reduce mistakes, improve communication and increase kitchen speed.
